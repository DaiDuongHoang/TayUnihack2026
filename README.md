# 👕 TayUnihack2026 – Taylr

## 🧠 Inspiration
People often choose their outfits without checking the weather conditions and may end up feeling uncomfortable or even facing health issues.  
We built this website to help users prepare properly for the weather and avoid unnecessary inconvenience in their daily life.

---

## 📌 Project Overview
**Taylr** is a smart wardrobe web application that helps users decide what to wear based on weather conditions.  

The system allows users to select their location, retrieve weather information, and recommend suitable outfits using the clothes stored in their digital wardrobe.  

The goal of this project is to make daily outfit planning easier by combining weather data with clothing preferences, creating a more convenient and personalized experience.

---

## ✨ Features
- 👔 Outfit recommendation
- 🌦 Weather API integration
- 🧥 Wardrobe viewing system
- 🔐 User authentication

---

## 🛠 Tech Stack
- YOLOv8
- OpenWeather API
- Streamlit
- Python

---

## ⚠️ Challenges
At the beginning, we faced several issues while installing required libraries and setting up the environment.  
We also struggled with backend programming since none of us had prior experience building a web application.  

Some features, such as camera integration and user authentication, took more time than expected.  
For authentication, we implemented Google login and used password hashing to improve security.

---

## 📚 What We Learned
Through researching and working on this project, our team gained a better understanding of how web applications are built and how different components work together.  

We also learned the importance of teamwork, as each member was responsible for a specific part of the project and contributed to the final result.

---

## 🚀 What's Next?
In the future, we plan to develop Taylr into a mobile application.  
We believe many users prefer taking photos and managing their wardrobe using their phones, so a mobile version would make the system more convenient and practical.
