# 👕 Taylr – Your Smart Wardrobe Assistant

## 🧠 Inspiration
Many people choose outfits without considering the weather, leading to discomfort or health issues. We built **Taylr** to bridge the gap between your closet and the forecast, ensuring you're always prepared for the day ahead.

---

## 📌 Project Overview
**Taylr** is an intelligent wardrobe web application that automates outfit selection based on real-time weather data. 

By integrating location-based weather tracking with a digital wardrobe, Taylr recommends the most suitable attire from your personal collection, streamlining your morning routine.

---

## ✨ Key Features
- 👔 **Smart Recommendations:** AI-driven outfit suggestions tailored to current conditions.
- 🌦 **Real-Time Weather:** Seamless integration with the OpenWeather API.
- 🧥 **Digital Wardrobe:** A visual system to manage and view your clothing collection.
- 🔐 **Secure Access:** User authentication featuring Google Login and industry-standard password hashing.

---

## 🛠 Tech Stack
- **Computer Vision:** YOLOv8 (Object Detection for clothing)
- **APIs:** OpenWeather API
- **Frontend/Backend:** Streamlit & Python
- **Security:** Bcrypt (Password Hashing) & OAuth

---

## ⚠️ Challenges & Solutions
* **Environment Setup:** We overcame initial hurdles with library dependencies by implementing strict environment management.
* **Full-Stack Development:** As a team of first-time web developers, we navigated the complexities of backend logic and state management in Streamlit.
* **Feature Integration:** Implementing secure authentication and live camera feeds required deep dives into API documentation and asynchronous programming.

---

## 📚 Lessons Learned
This project was a crash course in full-stack development. We gained hands-on experience in:
* Integrating machine learning models into web interfaces.
* Managing API data flow.
* **Agile Collaboration:** Using GitHub for version control and coordinating specific roles within a five-person team to meet the Unihack deadline.
